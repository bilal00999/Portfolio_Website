Blog Summariser a smart web app that summarizes blog articles and translates them into Urdu in seconds!
What it does:
Fetches any blog from a URL
Extracts and summarizes the main content
Translates the summary into Urdu using MyMemory API
Stores summaries in Supabase
Stores full blog content in MongoDB

Technologies Used: Next.js App Router, Supabase, MongoDB Atlas, Cheerio + Axios, MyMemory Translation AP

Github : https://github.com/bilal00999/Blog-Summariser

Live Preview: https://blog-summariser-black.vercel.app/

Description:

AI Recipe Generator a smart web app that creates personalized recipes based on the ingredients you already have.
How it works:
Just enter a few ingredients hit "Generate Recipe" and the app uses AI (Gemini API via n8n) to create a complete recipe including title, cooking time, servings, and step-by-step instructions.

Technologies Used: React.js + Vite, Tailwind CSS + ShadCN UI, Node.js + Express, Supabase for authentication, MongoDB for storing recipe history, n8n + Gemini API for AI logic
